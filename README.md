## 1. 난임이란?

> **피임 없이 정상적인 부부생활을 1년 이상 해도 임신이 되지 않는 상태**

### 주요 원인

| 구분 | 원인 | 비율 |
|------|------|------|
| **여성 요인** | 배란 장애(PCOS 등), 난관 손상·폐색, 자궁내막증, 자궁 구조 이상, 난소 기능 저하(AMH↓), 고령으로 인한 난자 질 저하 | ~50% |
| **남성 요인** | 정자 수 감소, 정자 운동성 저하, 정자 기형률 증가, 정계정맥류, 호르몬 이상 | ~40% |
| **원인불명** | 검사상 이상 없음 | ~10% |

### 핵심 검사 지표: AMH (항뮬러관 호르몬)

- 난소 예비력(남아있는 난자 수)을 가장 잘 반영하는 혈액 지표
- 나이가 같아도 AMH가 낮으면 난임 위험 높고, 시술 시 과배란 반응도 낮게 나타남

---

## 2. 노산이란?

> **만 35세 이상에서의 임신·출산**

35세를 기점으로 난소 내 난자 수와 난자의 질이 급격히 저하되기 시작합니다.

### 나이별 자연임신 확률 (월 기준)

| 나이 | 월별 자연임신 확률 |
|------|-----------------|
| 35세 | 약 15~20% |
| 40세 | 약 5% |
| 43세 이상 | 약 1~2% |

### 노산 주요 위험 요소

| 위험 요소 | 내용 |
|-----------|------|
| **염색체 이상 증가** | 다운증후군 위험: 35세 1/350 → 40세 1/100 → 45세 1/30 |
| **산과적 합병증** | 임신성 당뇨, 임신중독증, 제왕절개율 증가, 유산율 증가 |
| **임신 성공률 저하** | 위 나이별 자연임신 확률 참고 |

---

## 3. 난임과 노산의 관계

노산(35세 이상) 자체가 난임의 **주요 원인** 중 하나로 분류됩니다.

- 난자의 수와 질이 함께 감소
- 수정된 배아의 염색체 이상 확률 증가
- 착상 실패율 상승
- **→ 시간이 중요하기 때문에, 노산 난임의 경우 초기부터 적극적 시술(IVF)로 진행하는 경우가 많음**

---

## 4. 난임 치료 단계

```
1단계 → 2단계 → 3단계
배란유도   인공수정   체외수정
(약물)     (IUI)      (IVF)
```

### 단계별 설명

**1단계 — 배란유도**
- 배란 장애가 있는 경우 약물로 배란을 유도
- 자연임신 시도

**2단계 — 인공수정 (IUI, Intrauterine Insemination)**
- 남편/파트너의 정자를 처리 후 자궁 내 직접 주입
- 경증 남성 난임, 자궁경관 요인 등에 사용

**3단계 — 체외수정 (IVF, In Vitro Fertilization)**
- 과배란 유도 → 난자 채취 → 체외 수정 → 배아 이식
- 가장 적극적인 시술로, 성공률이 가장 높음
- 노산 난임의 경우 처음부터 IVF로 바로 진행하는 경우도 많음

### DI (Donor Insemination) — 공여자 정자 인공수정

일반 IUI와 달리 **익명 공여자의 정자**를 사용하는 시술

| 구분 | IUI (일반 인공수정) | DI (공여자 정자 인공수정) |
|------|-------------------|------------------------|
| 정자 출처 | 남편/파트너 | 익명 정자 공여자 |
| 대상 | 남성 경증 난임 | 무정자증, 중증 남성 난임, 유전질환 보유 |
| 정자은행 | 사용 안 함 | 정자은행 이용 (국내: 대한산부인과학회) |

## 5. 시술 코드 설명

해커톤 데이터에 등장하는 `TR`로 시작하는 코드는 **건강보험심사평가원(HIRA) 생식보조술 행위 분류 코드**로 추정됩니다.

| 코드 | 추정 시술명 | 설명 |
|------|------------|------|
| `TRCMWS` | 배아 이식 (Embryo Transfer) | 체외수정 후 수정란을 자궁에 이식 |
| `TRDQAZ` | 난자 채취 (Oocyte Retrieval) | 과배란 유도 후 난자 채취 |
| `TRJXFG` | 체외수정 (IVF) | 체외에서 난자와 정자를 수정 |
| `TRVNRY` | 배아 동결 (Embryo Cryopreservation) | 잉여 배아 냉동 보존 |
| `TRXQMD` | 동결배아 이식 (FET) | 동결 배아 해동 후 이식 |
| `TRYBLT` | 정자 처리 (Sperm Preparation) | 인공수정·IVF를 위한 정자 선별·처리 |
| `TRZKPL` | 착상 전 유전자 검사 (PGT) | 이식 전 배아 유전자 이상 여부 검사 |

> ⚠️ **주의**: 코드 정의는 패턴 기반 추정입니다. 정확한 정의는 제공된 코드북(Codebook) 또는 [HIRA 홈페이지](https://www.hira.or.kr)에서 확인하세요.

### IVF 시술 흐름과 코드 연결

```
과배란 유도
    ↓
난자 채취 (TRDQAZ)
    ↓
정자 처리 (TRYBLT)
    ↓
체외수정 (TRJXFG)
    ↓
배아 배양
    ↓
착상 전 유전자 검사 (TRZKPL) ← 선택적
    ↓
배아 이식 (TRCMWS)  또는  배아 동결 (TRVNRY)
                                  ↓
                         동결배아 이식 (TRXQMD)

## 대회 개요

난임 환자 데이터를 분석하여 임신 성공 여부를 예측하는 AI 모델을 개발합니다.

- **평가 지표**: ROC-AUC (Test 데이터 100% 기준)
- **일 최대 제출**: 3회

```

```
