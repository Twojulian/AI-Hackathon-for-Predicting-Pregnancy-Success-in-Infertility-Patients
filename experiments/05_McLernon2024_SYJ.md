# 🧬 IVF 임신 성공 예측 모델 — SYJ

> 불임 시술 데이터를 기반으로 임신 성공 여부를 예측하는 머신러닝 모델

---

## 📚 참고 논문

### 1. Shingshetty et al. (2024) — 피처 엔지니어링 근거
**"Predictors of success after in vitro fertilization"**
*Fertility and Sterility, 121, 742–751*

IVF 성공에 영향을 미치는 주요 예측 변수를 분석한 논문으로, 아래 피처 설계의 근거로 활용:
- 이전 임신/출산 이력이 성공률에 미치는 영향
- 시술 횟수와 임신 성공률의 관계
- 나이(특히 38세 이상)에 따른 성공률 감소

### 2. Alson et al. (2026) — 모델 설계 참고
**"Machine learning prediction of live birth after IVF using the morphological uterus sonographic assessment group features of adenomyosis"**
*Scientific Reports, 16:4324*

XGBoost 기반 IVF 생아 출산 예측 모델 논문으로, 아래 사항을 참고:
- Optuna를 활용한 하이퍼파라미터 최적화 방법론
- Stratified 5-fold cross-validation 구조
- SHAP 기반 피처 중요도 해석 방법
- 모델 성능 평가 기준 (AUC = 0.66~0.75)

## 📊 주요 발견

### Feature Importance 분석
1. **`이식된 배아 수`** (0.389) — 단일 피처 압도적 지배
2. **`저장된 배아 수`** (0.108)
3. **`배아 이식 경과일_결측여부`** (0.101) — 추가 피처 효과 확인
4. **`배아 이식 경과일`** (0.074)
5. **`시술 당시 나이`** (0.062)
6. **`고령_여부`** (0.053) — 추가 피처 효과 확인

엔지니어링 피처 중 `고령_여부`, `난자_배아_전환율`, `배아_사용_조합`이 Top 15 진입.

### 현재 문제점
- Class 1 (임신 성공) precision 0.39로 낮음 → 클래스 불균형 영향
- 클래스 비율: 실패(0) 38,025건 vs 성공(1) 13,246건 (약 3:1)
- `이식된 배아 수` 단일 피처 의존도 과다
