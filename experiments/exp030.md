# exp030 — 4모델 앙상블 (LGB + CAT + XGB + ExtraTrees)

## 실험 개요

| 항목 | 내용 |
|------|------|
| 실험 번호 | exp030 |
| 기반 실험 | exp024 피처 구성 + Optuna 파라미터 |
| 모델 | LightGBM + CatBoost + XGBoost + ExtraTrees |
| 앙상블 방법 | Rank Average (3모델 vs 4모델 비교) |
| 검증 방법 | Stratified 5-Fold OOF |
| 핵심 질문 | ExtraTrees 추가로 앙상블 다양성을 확보하면 AUC가 올라갈까? |

---

## 실험 배경 및 목적

exp029까지 앙상블 조합 방식(Stacking)을 바꿔봤지만 exp024(0.74025) 벽을 넘지 못했다. LGB/CAT/XGB 세 모델의 예측값이 너무 비슷한 것이 한계로 분석됐다.

이를 해결하기 위해 **분기점을 랜덤으로 정하는 ExtraTrees를 추가**해 앙상블 다양성을 확보하는 방향을 시도했다. ExtraTrees는 LGB/CAT/XGB와 학습 방식이 달라 다른 패턴을 포착할 것으로 기대했다.

---

## ExtraTrees 선택 이유 (vs RandomForest)

| 모델 | 특징 | 다양성 |
|---|---|---|
| ExtraTrees | 분기점을 **랜덤**으로 결정 | LGB/CAT/XGB와 패턴 차이 큼 |
| RandomForest | 분기점을 **최적값**으로 결정 | LGB/XGB와 학습 방식 유사 |

RandomForest보다 ExtraTrees가 앙상블 다양성 확보에 더 적합하다고 판단했다.

## 결론 및 인사이트

**ET 추가가 오히려 앙상블 성능을 낮췄다.** 원인은 명확하다.

ET 단독 AUC가 0.73600으로 LGB/CAT/XGB(0.7399~)보다 0.004 낮았다. 다양성은 확보됐지만 **성능이 낮은 모델을 앙상블에 넣으면 노이즈가 되어 발목을 잡는다.**

결과적으로 3모델 Rank Average(0.74025)가 자동 선택됐고 exp024와 동일한 값이 나왔다.

## 최종 결론

exp024~030까지 피처 추가/제거, Target Encoding, Stacking, 모델 추가를 전부 시도했지만 **exp024(0.74025)가 최종 베스트**다. 논문 벤치마크 범위(0.66~0.75) 내에서 현실적인 상한에 도달한 것으로 판단된다. AMH, AFC, BMI 등 핵심 임상 수치 부재가 근본적인 데이터 한계다.

→ **exp024로 최종 제출 확정.**  -> 0.7416940632
