# exp032 — Mixed FE v2 (팀원 방식 적용 + 파라미터 실험)

## 실험 개요

| 항목 | 내용 |
|------|------|
| 실험 번호 | exp032 (1차~4차 변형 실험 포함) |
| 기반 실험 | exp031 리크 문제 수정 + 팀원 EXP-032 방식 적용 |
| 모델 | LightGBM + CatBoost + XGBoost |
| 앙상블 방법 | Rank Average + Optuna 가중치 탐색 (300회) |
| 검증 방법 | Stratified 5-Fold OOF |
| 핵심 질문 | fold-wise TE/ITE + 팀원 파라미터 적용 시 AUC가 올라갈까? |

---

## exp031 대비 핵심 수정사항

### ① TE/ITE를 fold 안에서 계산 (리크 방지)
exp031은 TE를 학습 루프 밖에서 미리 계산해서 데이터 리크가 있었다. exp032는 **각 fold의 train 데이터로만 TE를 계산**해서 리크를 완전히 차단했다.

### ② ITE 방식 변경 (단순 곱셈 → 조합 TE)
exp031은 TE된 컬럼끼리 곱셈했지만, exp032는 **두 컬럼을 문자열로 합쳐서 조합 자체의 타겟 평균**을 계산한다.
```
# exp031 방식 (단순 곱셈)
TE(시술유형) × TE(나이)

# exp032 방식 (조합 TE)
"IVF_26" → 해당 조합의 임신 성공률
"IVF_41" → 해당 조합의 임신 성공률
```

### ③ FE-v2 추가 피처 4개
CAT/XGB에 v1 대비 4개 피처 추가: `임신_성공률`, `시술_실패_횟수`, `최적_난자수`, `나이_이식배아수`

### ④ Optuna 앙상블 가중치 탐색
Rank Average 고정 대신 LGB/CAT/XGB 최적 비율을 Optuna 300회로 탐색 (약 1~2분 소요)

### TE 적용 컬럼 (9개)

시술 당시 나이×시술 유형, 시술 당시 나이×특정 시술 유형, 시술 당시 나이×난자 출처, 시술 당시 나이×배란 유도 유형, 시술 유형×총 시술 횟수, 난자 출처×시술 유형

## 결론 및 인사이트

**exp032가 처음으로 exp024 OOF 벽(0.74025)을 넘었다.**

핵심 원인은 두 가지다. 첫째, fold-wise TE로 리크를 차단하면서 CAT/XGB 성능이 안정화됐다. 둘째, ITE(조합 TE)가 단순 곱셈 대비 비선형 패턴을 더 잘 포착했다.

Optuna 앙상블 가중치에서 XGB 비중이 가장 높게 나왔다(XGB≈0.5). ITE 피처가 XGB 성능 향상에 기여한 것으로 분석된다.

팀원 파라미터(EXP-019/011/012)가 우리 피처 구성과도 잘 맞았으며, 특히 CAT/XGB는 팀원 파라미터 적용 후 성능이 향상됐다.

→ **exp032 2차(팀원 파라미터 전체)로 최종 제출 시도.**
