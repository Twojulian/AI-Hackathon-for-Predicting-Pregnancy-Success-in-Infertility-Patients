# exp027 — 피처 선별 실험 (importance 기반 하위 피처 제거)

## 실험 개요

| 항목 | 내용 |
|------|------|
| 실험 번호 | exp027 (+ 027_1, 027_2 변형 실험 포함) |
| 기반 실험 | exp026 (전처리 + 파라미터 동일) |
| 모델 | LightGBM + CatBoost + XGBoost 앙상블 |
| 앙상블 방법 | Rank Average |
| 검증 방법 | Stratified 5-Fold OOF |
| 핵심 질문 | 하위 importance 피처를 제거하면 AUC가 올라갈까? |

---

## 실험 배경 및 목적

exp026에서 신규 피처 11개를 추가했음에도 exp024 대비 AUC가 소폭 하락했다 (0.74025 → 0.74016). 피처 수가 113개로 늘어나면서 모델이 핵심 변수에 집중하지 못하고 노이즈에 영향을 받았을 가능성이 제기됐다.

이를 검증하기 위해 **exp026의 3모델 Rank Average importance를 기준으로 하위 피처를 단계적으로 제거**하는 실험을 설계했다. importance는 LGB / CAT / XGB 각각의 순위를 평균내어 산출했다.

---

## 실험 흐름

### exp027 — 기준 실험 (81위 이하 30개 제거)

**제거 근거:** exp026 importance 순위에서 81위 이하 피처 전부 제거. 신규 피처 6개 + 기존 피처 24개.

**제거된 주요 피처:**

| 구분 | 피처 | 순위 |
|---|---|---|
| 신규 | 시술횟수_구간 | 86위 |
| 신규 | 시술유형_BLASTOCYST | 88위 |
| 신규 | 시술유형_IUI ~ ICI | 96~111위 |
| 기존 | 불임_원인_개수 | 82위 |
| 기존 | 정자 관련 원인 피처들 | 97~107위 |
| 기존 | 난자 채취 경과일, 신선_시술_여부 등 | 109~110위 |

**결과:**

| 지표 | 값 |
|---|---|
| OOF AUC | 0.74021 |
| vs exp026 | +0.00005 |
| 피처 수 | 81개 |

→ 피처 제거 효과 확인! 노이즈 제거로 소폭 상승.

---

### exp027_1 — 70위 이하 전부 제거 (40개 제거)

**제거 근거:** exp027에서 더 공격적으로 71~80위 피처 10개 추가 제거. 임상적 중요도 고려 없이 기계적으로 순위 기준만 적용.

**추가 제거된 피처 (71~80위):**

| 피처 | 순위 | 비고 |
|---|---|---|
| 이전_임신_여부 | 71위 | 논문 핵심 변수 |
| DI 임신 횟수 | 72위 | |
| 착상 전 유전 진단 사용 여부 | 73위 | |
| 동결 배아 사용 여부 | 74위 | |
| IVF_DI_시술_합산 | 75위 | |
| 총 시술 횟수 | 76위 | 임상적으로 중요 |
| 불명확 불임 원인 | 77위 | |
| 시술유형_ICSI | 78위 | 신규 |
| 시술유형_IVF | 79위 | 신규 |
| 기증자 정자와 혼합된 난자 수 | 80위 | |

**결과:**

| 지표 | 값 |
|---|---|
| OOF AUC | 0.74021 |
| vs exp027 | ±0.00000 |
| 피처 수 | 71개 |

→ 10개 더 제거했는데 AUC 변화 없음. 71~80위 피처들은 있으나 없으나 모델에 영향 없음.

---

### exp027_2 — 임상 중요 피처 보존 버전 (38개 제거)

**제거 근거:** exp027_1과 동일하되, 논문(Shingshetty et al., 2024)에서 핵심 변수로 꼽힌 **`총 시술 횟수`(76위)와 `이전_임신_여부`(71위)는 유지**. 데이터 기반 importance가 낮더라도 임상적 근거가 있는 변수는 살려야 한다는 가설 검증.

**결과:**

| 지표 | 값 |
|---|---|
| OOF AUC | 0.74019 |
| vs exp027_1 | -0.00002 |
| 피처 수 | 73개 |

→ 오히려 소폭 하락. 임상적으로 중요한 변수라도 **이 데이터에서는** 다른 피처들이 이미 해당 정보를 포착하고 있음을 시사.

## 전체 실험 결과 비교

| 실험 | OOF AUC | 피처 수 | vs exp024 |
|---|---|---|---|
| exp024 | 0.74025 | 102 | 기준 |
| exp026 | 0.74016 | 113 | -0.00009 |
| **exp027** | **0.74021** | **81** | **-0.00004** |
| exp027_1 | 0.74021 | 71 | -0.00004 |

---

## 특이사항

- 피처 선별 방향 자체는 맞았음 — exp026(113개) → exp027(81개)로 줄이니 AUC 회복
- 그러나 70위 이하로 더 공격적으로 제거해도 추가 향상 없음 → 71~80위 구간이 AUC의 임계점
- `총 시술 횟수`, `이전_임신_여부`는 논문에서 중요하지만 데이터 내에서는 `나이_시술횟수_상호작용`, `전체_임신_성공률` 등이 이미 해당 정보를 커버하고 있음

---

## 인사이트

- **피처 선별의 최적점은 약 80개** — 그 이상 줄여도 AUC 향상 없음
- 논문 기반 임상 중요도와 데이터 기반 importance가 항상 일치하지 않음. 이 데이터에서는 **배아 효율 관련 피처들**(이식률, 저장률, 수정률)이 가장 강력한 예측 변수
- 피처 선별만으로 exp024(0.74025) 돌파는 어려운 상황 → 다음 방향으로 타겟 인코딩 또는 새로운 전처리 접근 필요

---

## 데이터 한계

논문(Shingshetty et al., 2024 / Alson et al., 2026) 벤치마크 범위(0.66~0.75) 내에 위치. AMH, AFC, BMI 등 핵심 임상 수치 부재로 인해 0.74 언저리가 현실적인 상한에 가까울 수 있음.
