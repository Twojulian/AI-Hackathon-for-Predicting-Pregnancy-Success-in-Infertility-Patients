# exp025 — 신규 피처 엔지니어링 베이스라인

## 실험 개요

| 항목 | 내용 |
|------|------|
| 실험 번호 | exp025 |
| 기반 실험 | exp024 (전처리 동일) |
| 모델 | LightGBM 단일 모델 |
| 검증 방법 | Stratified 5-Fold OOF |
| OOF AUC | 0.73982 |

---

## 배경 — 왜 이 피처들인가?

**Shingshetty et al. (2024)** 논문에서 43개 IVF 예측 논문을 메타 분석한 결과, 임신 성공 예측에 가장 중요한 변수로 **여성 나이**, **불임 원인**, **배아 품질**, **시술 이력**을 꼽았다.

그러나 현재 데이터에는 AMH, AFC, BMI 등 핵심 임상 수치가 없어 직접 사용이 불가능하다. 따라서 **기존 변수들의 조합으로 논문의 핵심 인사이트를 간접적으로 반영**하는 파생 피처를 설계했다.

---

## 신규 파생 피처 5종

### 1. 나이 × 불임 원인 개수 (`나이_불임원인_상호작용`)
- **근거**: 논문에서 나이는 가장 강력한 단일 예측 변수. 고령일수록 복합 원인이 예후에 더 치명적임
- **설계**: `시술 당시 나이 × 불임_원인_개수`

### 2. 나이 × 이전 임신 여부 (`나이_이전임신_상호작용`)
- **근거**: 이전 임신 경험은 난소 반응성의 간접 지표. 고령이어도 이전 임신 이력이 있으면 예후가 다름
- **설계**: `시술 당시 나이 × 이전_임신_여부`

### 3. 시술 횟수 구간화 (`시술횟수_구간`)
- **근거**: 시술 횟수는 선형 관계가 아님. 1회(첫 시술), 2~3회(경험 축적), 4회 이상(반복 실패)로 패턴이 다름
- **설계**: `총 시술 횟수` → 0(1회) / 1(2~3회) / 2(4회 이상)

### 4. 해동 배아 비율 (`해동_배아_비율`)
- **근거**: 동결-해동 시술(FER)과 신선 시술의 성공률 차이를 반영
- **설계**: `해동된 배아 수 / 총 생성 배아 수`

### 5. 특정 시술 유형 이진 변수 분리 (`시술유형_IVF` 등 7개)
- **근거**: 시술 유형을 LabelEncoding만 하면 순서 관계가 생겨 모델이 잘못 학습할 수 있음
- **설계**: IVF / ICSI / IUI / FER / BLASTOCYST / GIFT / ICI 각각 이진 변수로 분리

## 논의

- 신규 피처 5종이 Top 20에 진입하지 못함 → 단일 LGB 베이스라인 파라미터에서는 효과가 제한적
- AUC는 exp024 기준선(0.73997) 대비 -0.00015로 사실상 동일한 수준
- **다음 방향**: exp024 Optuna 앙상블에 신규 피처를 결합하여 시너지 여부 확인 예정

---

## 특이사항

- `scale_pos_weight` 누락 후 재실험 진행 — 누락 시 Class 1 recall 0.13으로 급락, 추가 후 0.68로 회복
- 신규 피처보다 기존 배아 효율 피처(`이식률`, `수정률`, `저장률`)가 상위권 유지 → 배아 관련 변수가 이 데이터에서 가장 안정적인 예측 변수임을 재확인
- `이식된 배아 수` 단일 피처 의존도가 낮아짐 (이전 실험 1위 → 이번 7위) → 피처 다양성 개선

## 인사이트

- 신규 교호작용 피처(나이×불임원인, 나이×이전임신)가 Top 20 미진입 → 나이 효과가 이미 `시술 당시 나이` 단독으로 충분히 포착되고 있을 가능성
- 시술 횟수 구간화도 기여도 낮음 → `총 시술 횟수` 자체가 이미 모델에서 잘 처리되고 있는 것으로 보임
- **해동 배아 비율**, **시술 유형 이진 변수**는 데이터에 해당 컬럼이 없거나 희소해서 효과가 제한됐을 수 있음 → 컬럼 존재 여부 재확인 필요
- 결론적으로 이 데이터에서 피처 엔지니어링보다 **앙상블 다양성이나 하이퍼파라미터 튜닝**이 AUC 향상에 더 효과적일 가능성이 높음

## 데이터 한계

논문에서 핵심으로 꼽은 AMH, AFC, BMI가 현재 데이터에 없어 AUC 0.74 언저리가 현실적인 상한에 가까울 수 있음. 이는 **Alson et al. (2026)** 논문의 벤치마크(0.66~0.75)와도 일치하는 범위.


# exp026 — exp024 Optuna 최적 파라미터 + exp025 신규 피처 결합

## 실험 개요

| 항목 | 내용 |
|------|------|
| 실험 번호 | exp026 |
| 기반 실험 | exp024 (파라미터) + exp025 (신규 피처) |
| 모델 | LightGBM + CatBoost + XGBoost 앙상블 |
| 앙상블 방법 | Rank Average |
| 검증 방법 | Stratified 5-Fold OOF |
| OOF AUC | 0.74016 |
| 기준선 (exp024) | 0.74025 |

---

## 실험 목적

exp025에서 설계한 신규 파생 피처 5종이 단일 LGB 베이스라인에서는 효과가 제한적이었다. exp024의 Optuna 튜닝된 3모델 앙상블 구조에서 신규 피처를 결합하면 시너지가 날지 검증하는 것이 목적이다.

## 특이사항

- 전체 피처 수 102 → 113으로 증가 (신규 11개 추가)
- exp024 대비 scale_pos_weight가 정상 적용됨 (exp024는 누락 상태였음)
- 3가지 앙상블 방법이 모두 0.74015~0.74016으로 수렴 → 모델 간 예측 패턴이 매우 유사함

---

## 인사이트

- 신규 피처 11개 추가에도 불구하고 exp024 대비 **-0.00009** 소폭 하락 → 신규 피처가 시너지보다 노이즈로 작용했을 가능성
- 피처 수 증가가 모델이 핵심 변수에 집중하는 것을 방해했을 수 있음
- **현재 베스트는 exp024 (OOF 0.74025)**
- 다음 방향으로 신규 피처 중 importance 낮은 것을 제거한 선별 버전 또는 타겟 인코딩 등 새로운 전처리 접근을 고려할 필요가 있음
