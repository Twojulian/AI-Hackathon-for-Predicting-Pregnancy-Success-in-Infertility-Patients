# exp031 — Mixed FE (모델마다 다른 피처셋)

## 실험 개요

| 항목 | 내용 |
|------|------|
| 실험 번호 | exp031 |
| 기반 실험 | exp024 피처 구성 + Optuna 파라미터 |
| 모델 | LightGBM + CatBoost + XGBoost |
| 앙상블 방법 | Rank Average |
| 검증 방법 | Stratified 5-Fold OOF |
| 핵심 질문 | 모델마다 다른 피처셋을 쓰면 앙상블 다양성이 높아져 AUC가 올라갈까? |

---

## 실험 배경 및 목적

exp024~030까지 피처 구성, 인코딩, 앙상블 방식, 모델 추가를 전부 시도했지만 exp024(0.74025) 벽을 넘지 못했다. 타 팀 실험(EXP-032)에서 **모델마다 완전히 다른 피처셋을 쓰는 Mixed FE** 방식으로 성능을 올린 사례를 참고해 시도했다.

---

## 모델별 피처셋 구성

| 모델 | 피처 수 | 구성 방식 |
|---|---|---|
| LGB | 81개 | LabelEncoding + importance 상위 81개 자동 선택 |
| CAT | 102개 | OOF Target Encoding (smoothing=10) |
| XGB | 106개 | OOF Target Encoding + 상호작용 피처 4개 추가 |

### XGB 상호작용 피처 (TE된 컬럼끼리 곱셈)
- 시술 유형 × 배아 이식 경과일
- 시술 유형 × 시술 당시 나이
- 난자 출처 × 시술 유형
- 정자 출처 × 시술 유형

| 지표 | 값 |
|---|---|
| LGB OOF AUC | 0.73981 |
| CAT OOF AUC | 0.73970 |
| XGB OOF AUC | 0.73883 |
| **앙상블 OOF AUC (Rank Average)** | **0.73997** |
| vs exp024 | -0.00028 |

## 전체 실험 결과 비교

| 실험 | OOF AUC | vs exp024 | 핵심 변경 |
|---|---|---|---|
| exp024 | **0.74025** | 기준 | Optuna 3모델 앙상블 |
| exp026 | 0.74016 | -0.00009 | 피처 11개 추가 |
| exp027 | 0.74021 | -0.00004 | 피처 81개로 축소 |
| exp028 | 0.73974 | -0.00051 | Target Encoding |

---

## 결론 및 인사이트

**Mixed FE도 exp024를 넘지 못했다.** 원인은 두 가지다.

첫째, XGB가 0.73883으로 세 모델 중 가장 낮았다. TE + 상호작용 피처 4개가 시너지가 아닌 노이즈로 작용했다. 데이터가 작을수록 피처를 늘리면 과적합 위험이 커진다는 패턴이 이번에도 반복됐다.

둘째, CAT도 TE 적용 후 exp024 CAT(0.73994)보다 낮은 0.73970이 나왔다. CatBoost는 내부적으로 이미 TE를 수행하기 때문에 외부 TE가 중복 처리가 되어 오히려 방해가 된 것으로 분석된다.

LGB만 importance 기반 피처 축소로 0.73981을 기록하며 선방했지만 나머지 두 모델이 발목을 잡았다.

---

## 최종 결론

exp024~031까지 피처 추가/제거, Target Encoding, Stacking, 모델 추가, Mixed FE를 전부 시도했으나 **exp024(0.74025)가 최종 베스트**. AMH, AFC, BMI 등 핵심 임상 수치 부재로 인한 데이터 한계가 근본 원인으로 판단된다.

→ **exp024로 최종 제출 확정.**
